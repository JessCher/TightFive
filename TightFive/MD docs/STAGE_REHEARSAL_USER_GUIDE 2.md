## Understanding Confidence Scores

**What they mean:**
- **80-100%**: Excellent - phrase will work on stage
- **60-79%**: Good - should work reliably
- **40-59%**: Okay - might work, consider improving
- **0-39%**: Poor - definitely needs better phrase

**If scores are low:**
1. Make phrases more distinctive (avoid common words)
2. Keep phrases 4-8 words long
3. Use phrases with unique rhythm/cadence
4. Avoid phrases that sound like other parts of your bit

## Common Issues

### "Auto-advance isn't working"
**Possible causes:**
- Exit phrase too generic
- Speaking too fast/unclear
- Background noise interfering
- Sensitivity set too high

**Solutions:**
- Check exit phrase confidence score
- Slow down delivery slightly
- Move to quieter location
- Lower sensitivity in Stage Mode Settings

### "Audio level stuck at 0%"
**Possible causes:**
- Microphone permission denied
- Wrong mic selected
- Mic hardware issue

**Solutions:**
- Check Settings â†’ Privacy â†’ Microphone
- Try different microphone
- Restart the app

### "Anchor phrase never detected"
**This is actually okay!** Anchor phrases are optional confirmation. As long as EXIT phrases work, you're good.

**But if you want better anchor detection:**
- Use more distinctive opening words
- Increase anchor sensitivity in settings
- Make sure you're saying the exact phrase

## Troubleshooting

### Analytics shows "0 transcriptions received"
- Speech recognition didn't start
- Check microphone permission
- Try restarting rehearsal

### All confidence scores are 0%
- You might not be saying your phrases
- Check that you're reading the actual script
- Verify phrases exist in setlist

### Too many manual transitions
- Phrases too generic
- Need more distinctive words
- Try 4-8 word phrases with unique rhythm

### Recognition works in rehearsal but not on stage
- Venue is louder than practice space
- Lower sensitivity slightly
- Use more distinctive phrases
- Practice in noisier environment
