## Using Your Settings in Stage Mode

### Launch Stage Mode
1. From the setlist builder, tap **â‹¯** â†’ **Stage Mode**
2. Optional: Enter a venue name
3. Tap **Begin Performance**

### During Performance
Your settings will be active:
- **Font size** affects all cue card text
- **Text color** changes card text color
- **Line spacing** adjusts readability
- **Auto-advance** determines if cards advance automatically
- **Phrase feedback** shows recognition confidence bars

### Manual Controls (Always Available)
Even with auto-advance enabled:
- **Swipe Left**: Next card
- **Swipe Right**: Previous card
- **Tap Auto/Manual**: Toggle auto-advance mid-performance
- **Tap X**: End performance and save recording

## Tips

1. **Test Before Performance**: Try different settings in rehearsal mode
2. **Lighting Matters**: Yellow text works best in bright stage lights
3. **Content Length**: Shorter cards = larger font, longer cards = smaller font (automatic)
4. **Manual Override**: Auto-advance ON doesn't disable manual swipes
5. **Save Presets**: Note your favorite settings for different venues

## Support

If Stage Mode doesn't launch:
1. Ensure setlist is marked "Finished" (not "In Progress")
2. Verify setlist has script content (script blocks or bits)
3. Grant microphone permissions in iOS Settings
4. Try restarting the app

Settings are stored locally and persist across app launches. No internet connection required!
